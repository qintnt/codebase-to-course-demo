这是什么？

一个能读懂代码、自动生成精美交互式课程的 AI 技能 —— 让任何人都能看懂软件是怎么运行的。

想象一下

你在 GitHub 上发现了一个很酷的项目。它跑得很好，但看代码就像看天书。

这个技能就是来解决这个问题的。它会读取整个代码库，搞清楚各部分是怎么连在一起的，然后生成一个精美的交互式课程——带有动画、测验和通俗易懂的代码翻译。

第三阶段：串行 vs 并行

这里是聪明之处。技能根据复杂度选择策略：

SKILL.md 
简单代码库 → 串行模式
  逐个章节写

复杂代码库 → 并行模式
  先写章节摘要（briefs）
  分配给多个子代理
  每批最多 3 个同时写
大白话

小而简单的项目，AI 直接按顺序逐个写章节，边写边检查。

大而复杂的项目，AI 先给每个章节写摘要，然后派多个子代理并行写，互不干扰。

摘要里提前摘录了代码片段，所以写章节的代理不需要再读代码库——省了大量 token。

设计哲学

温暖的视觉、丰富的比喻、通俗易懂。一切都是为从没看过代码的人设计的。

先做，再理解

这个技能颠覆了传统教育。不是先学几年理论再碰代码，而是从一个已经能跑的东西开始——然后一层层剥开看。

🏗️
比喻

就像拆开一块手表。你已经知道它能看时间，现在打开表盖看齿轮——每一块都在上下文里有意义，因为你知道整体是干什么的。

❌

旧方式
背概念 → 也许能做出东西 → 大多数人放弃了

vs
✅

这个技能
先做出东西 → 看到它跑 → 然后理解原理

展示，不要说教

每个页面至少 50% 是视觉内容。文字块最多 2-3 句。能做成图、动画、交互元素的，就不要写成一整段。

大段文字会让非技术读者望而生畏，看起来像作业。

视觉设计有很明确的偏好——拒绝千篇一律的紫色渐变：

🌅

暖色调

偏米色的背景像旧纸张，暖灰色。感觉像程序员的笔记本。

✏️

Bricolage Grotesque 做标题——大胆有几何感。绝不用 Inter 或 Roboto。

🎯

一个强调色

朱红、珊瑚、蓝绿、琥珀、森林——只选一个自信的强调色。

Catppuccin 风格的语法高亮，深靛蓝色背景。感觉像真正的 IDE。

05

交互引擎

一个 14KB 的 JavaScript 文件驱动所有测验、动画和交互。零框架、零依赖。

一个文件搞定一切

整个交互体验都由 main.js 驱动。没有 React、没有 Vue，就是原生 JavaScript，加上智能的自动检测：

📍

自动检测

扫描页面上的类名（.chat-window、.flow-animation 等），自动激活对应的引擎。

🎬

滚动动画

IntersectionObserver 在滚动时渐入内容，支持子元素交错动画。

📖

词汇浮窗

position: fixed 浮窗，挂在 document.body 上，永远不会被容器的 overflow: hidden 裁掉。

📱

移动端适配

拖拽支持触摸事件，响应式网格布局，全面移动端优先。

自动检测是怎么做到的

技能用 querySelectorAll 按类名找组件，自动绑上交互：

main.js 
// 找到所有聊天窗口
$$('.chat-window')
  .forEach(el => initChat(el));

// 找到所有流程动画
$$('.flow-animation')
  .forEach(el => initFlow(el));

// 找到所有拖拽容器
$$('.dnd-container')
  .forEach(el => initDnD(el));
