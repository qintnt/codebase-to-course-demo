SKILL.md — The Brain

The main instruction file is 17KB of detailed guidance. It tells the AI:

1

Trace data flows, identify components, map how everything connects

2
Phase 2: Design the Curriculum

Structure 4-6 modules from user-facing behavior down to internals

3
Phase 3: Build the Course

Write module HTML using the reference files — never regenerate CSS/JS

4
Phase 4: Assemble and Review

Run build.sh, open the result, check for quality

See How It Works

Here's a real snippet from build.sh — it's just 4 lines, but it's the whole assembly process:

build.sh 
#!/bin/bash
set -e
cat _base.html modules/*.html \
  _footer.html > index.html
echo "Built index.html"
PLAIN ENGLISH

Run this as a shell script.

Stop immediately if any command fails.

Take the HTML shell, all module files, and the footer, and glue them together...

...into a single index.html file.

Print a success message.
