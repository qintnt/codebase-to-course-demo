文件里面有什么

两大块：指令（SKILL.md）和原材料（references/）。就像食谱 + 厨房。

整体结构

这个技能的结构像一个组件库：一个主指令文件，加一个装满可复用零件的文件夹。

codebase-to-course/ 技能根目录
SKILL.md 主指令——"食谱"本身
references/ 可复用的零件
styles.css 完整设计系统——600+ 行 CSS
main.js 交互引擎——测验、聊天、动画
_base.html HTML 外壳模板，含占位符
build.sh 一行命令拼出 index.html
interactive-elements.md 每种交互组件的 HTML 写法
design-system.md CSS 变量、配色、排版指南
gotchas.md 常见坑，发布前必查

SKILL.md — 大脑

主指令文件有 17KB 的详细指南，告诉 AI：

1
第一阶段：读代码

追踪数据流、识别组件、理清各部分怎么连在一起

2
第二阶段：设计课程

规划 4-6 个章节，从用户体验逐步深入到内部实现

3
第三阶段：写课程

用参考文件写模块 HTML——绝不要重新生成 CSS/JS

4
第四阶段：组装和检查

运行 build.sh，打开结果，检查质量
